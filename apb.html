Alan's Psychedelic Breakfast
Audio Processing and Visualization
Fall '14
Alan's Psychedelic Breakfast is an real-time audio visualizer which analyzes the audio it can hear to generate "trippy" psychedelic visuals. Not only can you listen to your music, but now you can see it too!
Warning: Extremely addictive and can result in the user staring at the screen for extended durations of time.
Project URL: https://ccrma.stanford.edu/~trijeetm/alan's-psychedelic-breakfast/
Audio processing in C/C++ using Rt-Audio library, visualization via OpenGL.
